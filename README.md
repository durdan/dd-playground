# DD Playground

DD Playground is an innovative project aimed at providing a comprehensive environment for software development lifecycle simulation. It integrates various aspects of development, testing, deployment, and monitoring into a seamless workflow.

## Project Overview

This project incorporates a unique language graph-based approach to model and visualize the software development lifecycle (SDLC). It offers a versatile platform for experimenting with different phases of development, from planning through deployment.

## Getting Started

To get started with DD Playground, follow these instructions:

1. Clone the repository:
   ```bash
   git clone https://github.com/durdan/dd-playground.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
