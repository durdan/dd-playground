# DD-Playground

## Overview

DD-Playground is a comprehensive toolkit designed to facilitate the development, testing, and deployment of software projects. It incorporates various tools and visualizers to streamline the entire software development lifecycle.

## Setup Instructions

To get started with DD-Playground, follow these steps:

1. Clone the repository: `git clone https://github.com/durdan/dd-playground.git`
2. Navigate to the project directory: `cd dd-playground`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`

## API Documentation

The project includes several APIs to interact with different parts of the software development process. For detailed information on available endpoints and their usage, refer to the `app/api/` directory.

## Usage Examples

Refer to `EXAMPLES.md` for sample pipeline runs and scenarios demonstrating the capabilities of DD-Playground.

## Troubleshooting

If you encounter any issues during setup or while using the application, please check the following:

- Ensure all dependencies are correctly installed.
- Verify that the development server is running and accessible.
- Check the `.env.local.example` file for required environment variables and configure them as needed.

## Environment Setup Guide

For setting up your environment, refer to the `.env.local.example` file. This file contains necessary environment variables configuration instructions to ensure the application runs correctly.
