# DD Playground

DD Playground is an experimental project exploring the capabilities and integrations of various development tools and technologies in a cohesive workflow. It aims to serve as a sandbox for testing, development, and continuous integration/delivery (CI/CD) processes.

## Getting Started

To get started with DD Playground, clone the repository and install the required dependencies:

```bash
cd dd-playground
git clone https://github.com/durdan/dd-playground.git
npm install
```

## Features

- Comprehensive set of components for UI development
- Integrated Git connection and management tools
- Advanced pipeline and workflow visualization
- Extensive testing and documentation support

## Usage

To run the project locally, execute the following command:

```bash
npm start
```

This command starts the development server, making the project accessible via `http://localhost:3000`.

For detailed information on project setup, development practices, and more, refer to the `EXAMPLES.md` and `CLAUDE.md` documents.
