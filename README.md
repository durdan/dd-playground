# DD Playground

DD Playground is a versatile toolkit designed to enhance software development workflows. It integrates various components, including workflow visualization, Git connection management, and more, to provide a comprehensive environment for developers.

## Features

- Workflow Visualization: Offers a clear view of the software development process.
- Git Connection Panel: Simplifies the management of Git repositories and branches.
- CLI Progress Panel: Displays real-time progress of command-line operations.

## Getting Started

To get started with DD Playground, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/durdan/dd-playground
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
